# Maceration Controller

Arduino-based controller for maceration processes with 4-channel relay control, passive buzzer alerts, speed sensor monitoring, and DS18B20 temperature monitoring.

## Hardware

- ESP32 / Arduino-compatible board
- 4-channel relay module (active-LOW)
- 4 tactile push buttons
- Passive buzzer
- Speed sensor
- DS18B20 temperature sensor

## Libraries

- [OneWire](https://github.com/PaulStoffregen/OneWire)
- [DallasTemperature](https://github.com/milesburton/Arduino-Temperature-Control-Library)
